# Automation testing of the Telnyx website with webdriver.io, Cucumber and Allure Reporter


## Versions
- Node JS - 18.14.2
- NPM - 9.5.0
- webdriver.io - 7.30.2


## How to run on your local machine
1. Clone this repo by using command in ```Git```.
```
git clone https://github.com/ddpolianskyi/telnyx-wdio-cucumber.git
```
2. Install all requirement dependencies by using command in ```cmd```.
```
npm install
```
3. Run tests by using command:
```
npm run test
```
4. Open Allure HTML report:
```
npm run allure:open
```
